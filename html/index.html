How to replace it with your own frontend
  1. Build your SPA (for example, run npm run build).
  2. Copy the build output (usually the dist/ or build/ folder) into /hidden/html.
  3. Make sure that /hidden/html contains index.html and all static assets (JS/CSS/images).
